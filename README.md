# Bootcamp Santander 2025 - Fundamentos de Lógica de Programação (FLP)

Este repositório reúne os **projetos e desafios práticos** desenvolvidos durante o bootcamp **Santander 2025 – Fundamentos de Lógica de Programação**.

O foco do bootcamp é construir uma base sólida em **lógica de programação**, essencial para quem deseja atuar como desenvolvedor, cientista de dados ou em qualquer área da tecnologia.

## Conteúdos Trabalhados

- Pensamento Computacional  
- Lógica de Programação  
- Estruturação de Código  
- Funções e Objetos  
- Automação de Decisões com Dados
